## About

Next Season is a mobile application developed as part of the Mobile Device Programming course at IADE - Engenharia Informática (European University). The app allows users to browse a list of movies or series, view detailed information about each title, and check user reviews.

The implementation focuses on modern Android development practices using **Jetpack Compose** for declarative UI and **Kotlin** for robust application logic.

## Features

- **Content Browsing**: Explore a catalog of movies and TV shows.
- **Detailed Views**: View rich information about each title, including description, genre, and release date.
- **User Reviews**: See ratings and reviews from other users.
- **Modern UI**: Built entirely with Jetpack Compose, ensuring a responsive and smooth user experience.
- **Modular Components**: UI is organized into reusable components with individual previews.

## How to run

1. Clone the repository:

   ```bash
   git clone https://github.com/nycocado/next-season-task.git
   ```

2. Open the project in **Android Studio**.
3. Sync the project with Gradle files.
4. Select an Android Emulator or a physical device and click **Run**.
